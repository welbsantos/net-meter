# net-meter

Mac OS app to monitor inbound and outbound network traffic using [Electron](http://electron.atom.io).

Built with [photon](http://photonkit.com).

## Running

```sh
git clone https://github.com/welbsantos/net-meter.git
cd net-meter
npm install
npm start
```

## Packaging

```sh
npm run package-mac
```
